Kelebihan:
- Codingan jQuery lebih pendek dan simple dibanding vanilla js.

- jQuery merupakan js library yang termasuk mudah dipakai untuk membuat fungsi yang kompleks dengan baris coding yang sedikit.

- jQuery kompatibel dengan hampir seluruh browser

- jQuery lebih banyak diminati oleh developer web saat ini

Kekurangan:
- Vanilla js hampir 25x lebih cepat dibanding jQuery.

- Vanilla js tidak bergantung pada library dan kita mendevelop sendiri apa yang diperlukan

- Vanilla js gratis secara mutlak karena tidak memakai library lain dan framework lain

- Dalam sisi server hosting jika Request jQuery sangat banyak yang perlu ditangani, maka akan membebani server

$( "button" ).on( "click", function() {
  $( "p" ).append( "Started..." );
 
  $( "div" ).each(function( i ) {
    $( this ).fadeIn().fadeOut( 1000 * ( i + 1 ) );
  });
 
  $( "div" ).promise().done(function() {
    $( "p" ).append( " Finished! " );
  });
});

-> setelah button diklik, maka akan menjalankan function jquery step by step dulu, mulai dari p hingga div selesai maka function promise muncul ketika div selesai, menambahkan function append <p> pada html
